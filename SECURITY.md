# Security Policy

## Supported versions

The latest published release of `@glacient.tech/cli` receives security updates.
Older versions are not maintained — please upgrade to the latest version before
reporting an issue.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use one of the following private channels:

- Open a private advisory via GitHub's
  ["Report a vulnerability"](https://github.com/glacient/glacient-cli/security/advisories/new)
  flow, or
- Email **security@glacient.tech** with details.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce, or a proof of concept.
- The affected version(s).

We will acknowledge your report as soon as possible and keep you informed of the
progress toward a fix and disclosure. Please give us a reasonable window to
address the issue before any public disclosure.

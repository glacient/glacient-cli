## v0.1.1

Initial public release.

### New Features

- `glacient login` / `glacient logout` — authenticate and manage local credentials.
- `glacient whoami` — show the signed-in identity.
- `glacient capabilities` — discover and call CLI capabilities (`rpc`, `local`,
  `skill`, and `ref` types).
- `glacient config` — inspect and modify persisted CLI settings.
- `glacient version` — print the CLI version (JSON or text, per `--output`).
- JSON-first output for AI-agent use; defaults to JSON when stdout is not a TTY
  (`--output json|text` to override).
- Environment-variable configuration:
  - `GLACIENT_SERVER_URL` — override the server URL.
  - `XDG_CONFIG_HOME` — base directory for credentials and config (defaults to
    `~/.config`; files live under `glacient.tech/cli/`).
